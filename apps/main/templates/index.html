Welcome on Django Bike

Release 0.8

3 Nov 2020

Features

  • Server compression Files are now served using Gzip compression. Your favourite website should load even faster than before!
  • Licensing This website is now developped under the MIT license

Fixes

  • Email templates Compiled email templates used to be unsuccessfully copied to correct location. This used to cause errors while sending emails to users
  • Alternate text for images This should improve accessibility for all users
  • Upgrade dependencies The following dependencies were upgraded for security reasons: node-fetch@2.6.1, cryptography@3.2.1

Release 0.7

16 Sep 2020

Features

  • Comments and Likes You can now like other people's tracks (if they are set to public), and comment them along your own tracks. Enjoy sharing your performances with the community!
  • Track state Cooler way to display a track state in the track list
  • Track admin search Tracks, Comments and Likes are searchable in the admin pages.
  • Forms Display an unified inline form anywere, and TextArea is available to use in code.
  • README Improved installation instructions.
  • Browsing Django Bike is available in HTTPS with HTTP/2, with this new domain name : bike.lehoux.net .

Fixes

  • Invalid headers Headers from the wrong origin are now denied to access Django.
  • Email templates build The email Javascript app used to fail the build stage.
  • Columns size Columns were not using the whole page width.
  • Track admin user ownership Saving a track would change the owner to the current admin user. It now does so only if there is no owner.
  • Delete track button It would show to anybody, now only for the owner (this was only a display issue).
  • Weekly and Monthly charts There was an error in the form to choose the time period, so none would show.
  • Delete notifications The delete button click wouldn't work.
